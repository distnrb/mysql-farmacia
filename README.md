# mysql-farmacia
### Criação de um banco de dados para um sistema de farmácias em três contextos:
1. WEB
2. FRENTE DE CAIXA
3. ESTOQUE

Possui:
- 20+ TABLES
- DIVERSAS FOREIGN KEYS
- INSERTS
- SELECTS
- VIEWS
- JOINS 
- STORED PROCEDURES
- FUNCTIONS
- TRIGGERS
